# '모바일앱제작(01)' 김명국 교수님 수업 실습 내용
- 꼰대테스트 앱

## Stacks
### Development
- React-Native
- JavaScript
### Environment
- Expo CLI(Expo Go)
- Visual Studio Code
- Github

## 시작 가이드
```
$ npx expo
```

## 앱 소개
- 꼰대력을 테스트할 수 있는 질문 10개에 대하여 “네/보통/아니요“ 중 하나를 선택합니다.
- “네”라고 대답한 개수에 따라 꼰대력의 정도를 “완전 꼰대“, “잠재적 꼰대“, “MZ”로 나눠 결과를 보여줍니다.
- 앱에 들어간 모든 그림은 직접 그렸습니다.
